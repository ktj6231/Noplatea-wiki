  • 자바 에디션 1.20.1 ~ 1.20.4

  1. 멀티플레이 선택
  2. 서버 추가 클릭
  3. 서버 주소: noplatea.world
  4. 완료 클릭

3단계: 첫 접속

  • 서버 선택 후 접속
  • 리소스팩 다운로드 허용

🎯 첫 접속 후 할 일

  1. 회원가입: /register 비밀번호 비밀번호
  2. 로그인: /login 비밀번호
  3. 스폰 지역 둘러보기
  4. 튜토리얼 NPC 찾기
  5. 레벨 5 달성 후 클래스 선택
